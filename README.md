# vene dash
   WordPress theme
   
## Tech stack
* WordPress 
* Gulp.js
* Stylus css
* jQuery
* Slicky slider
* WP plugin ACF ( Advanced custom fields ) 
  
### Highlight
Comparison page: created to compare vene dash with other marketing platforms
   
## Start local project
On terminal:

     $ npm install
     $ gulp
