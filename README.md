# vene dash

   WordPress theme created for vene dash - developed at freenet digital

# Tech stack
    *WordPress 
    *Gulp.js
    *Stylus css
    *jQuery
    *Slicky slider
    *WP plugin ACF ( Advanced custom fields ) 
  
# Highlight

   Comparison page: created to compare vene dash with other marketing platforms
